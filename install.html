Install ClawPy

Get started with ClawPy in just a few steps. Follow the instructions below to install the framework and its dependencies on your system.

1. Install Python

Ensure Python 3.8 or higher is installed. Download from python.org.

2. Install pip

Verify pip is installed: 

pip --version

If pip is not found, install it via pip's official guide.

3. Install ClawPy

Install ClawPy via pip: 

pip install clawpy

4. Install Dependencies

Install optional dependencies for full functionality: 

pip install matplotlib sympy numpy

5. Verify Installation

Confirm ClawPy is installed by checking its version: 

pip show clawpy

This should output the installed version with all other details.

6. Getting Started

Try a simple ClawPy example: 

from clawpy import BasicMath
cw = BasicMath()

print(cw.add(8,5))  # Should print 13
print(cw.multiply(7,8))  # Should print 56

This should output 13 and 56 respectively.

Troubleshooting

Pip Not Found

If you see a "pip not found" error, ensure pip is installed: 

python -m ensurepip --upgrade
python -m pip install --upgrade pip

Dependency Conflicts

If you encounter dependency issues, try upgrading pip or using a virtual environment. Alternatively, install a specific version: 

pip install clawpy==0.1
