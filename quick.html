Get Started with ClawPy: Unleash Your Math Potential! 🚀

Dive into ClawPy and explore its powerful modules for math and science. From orbits to AI, this guide offers step-by-step examples to kickstart your journey. Use the sidebar to navigate modules and begin!

1. ScientificMath: Physics & Chemistry

Tackle scientific calculations like heat transfer and orbital speeds with ScientificMath.

🔥Step 1: Calculate Heat Transfer

Compute energy needed to heat a substance.

from clawpy import ScientificMath
sci_math = ScientificMath()
print(sci_math.heat_transfer(2, 4.18, 10))  # 2kg, 4.18 J/g°C, 10°C

Energy for heating 2kg of water by 10°C.

🧪Step 2: Molecular Weight

Calculate the molecular weight of H₂O.

elements = ["H", "O"]
amounts = [2, 1]
print(sci_math.molecular_weight(elements, amounts))

Returns the molecular weight of water.

🌌Step 3: Orbital Velocity

Find the speed for Earth orbit.

earth_mass = 5.972e24  # kg
earth_radius = 6.371e6  # m
print(sci_math.orbital_velocity(earth_mass, earth_radius))

Calculates satellite orbital speed.

2. OtherMath: Number Patterns

Explore sequences and graphs with OtherMath.

🔢Step 1: Lucas Numbers

Generate Lucas number sequences.

from clawpy import OtherMath
other_math = OtherMath()
print(other_math.lucas_number(10))  # 10th Lucas number

Prints 123, the 10th Lucas number.

🕸️Step 2: Shortest Paths

Use Dijkstra’s algorithm for network routing.

graph = {'A': {'B': 1, 'C': 4}, 'B': {'A': 1, 'C': 2}, 'C': {'A': 4, 'B': 2}}
print(other_math.dijkstra(graph, 'A'))

Finds shortest paths from point A.

🎲Step 3: Estimate Pi

Approximate π with Monte Carlo.

print(other_math.monte_carlo_pi(1000))

Estimates π (~3.14).

3. MoreMath: Advanced Math

Handle matrices and transforms with MoreMath.

📏Step 1: N-Dimensional Distance

Measure distances in higher dimensions.

from clawpy import MoreMath as mm
p1 = [1, 2, 3]
p2 = [4, 5, 6]
print(mm.distance_in_nD(p1, p2))

Distance between two 3D points.

📡Step 2: Fourier Transform

Analyze signal frequencies.

import numpy as np
signal = [0, 1, 0, -1]
print(mm.fourier_transform(signal))

Shows frequency components.

🧮Step 3: Solve Equations

Solve equations like x² - 4 = 0.

import sympy as sp
x = sp.symbols('x')
equation = x**2 - 4
print(mm.solve_equation(equation, x))

Solutions: x = 2, -2.

4. LeftMath: Geometry & Patterns

Explore shapes and sequences with LeftMath.

🟠Step 1: Circle Area

Compute the area of a circle.

from clawpy import LeftMath
lm = LeftMath()
print(lm.circle_area(5))

Area of a circle with radius 5.

🐰Step 2: Fibonacci Numbers

Generate Fibonacci sequences.

print(lm.fibonacci(10))

Prints 55, the 10th Fibonacci number.

🪐Step 3: 4D Tesseract

Visualize a rotating 4D cube.

lm.draw_tesseract()

Draws a 4D tesseract (close window to stop).

5. Calculus: Derivatives & Integrals

Master calculus with the Calculus module.

📉Step 1: Derivative

Find the slope of a curve.

from clawpy import Calculus
calc = Calculus()
def f(x): return x**3 + 2*x**2 + x
print(calc.derivative(f, 2))

Slope of x³ + 2x² + x at x=2.

📍Step 2: Integral

Calculate area under a curve.

print(calc.integrate(f, 0, 2))

Area under curve from x=0 to x=2.

🔄Step 3: Differential Equations

Solve with Euler’s method.

def df(x, y): return x + y
print(calc.euler_method(df, x0=0, y0=1, h=0.1, steps=10))

Solves dy/dx = x + y.

6. AppliedMath: Real-World Math

Apply math to physics and visualization with AppliedMath.

✈️Step 1: Aerodynamic Lift

Compute lift for a wing.

from clawpy import AppliedMath
applied_math = AppliedMath()
print(applied_math.aerodynamic_lift(1.225, 30, 2, 0.5))

Calculates lift for an airplane wing.

📈Step 2: Plot a Function

Generate points for a parabola.

x, y = applied_math.generate_plot(lambda x: x**2)
print(x[:5], y[:5])

Creates points for x².

7. AI: Machine Learning Tools

Build AI models with the AI module.

📊Step 1: Model Accuracy

Check prediction accuracy.

from clawpy import AI
import numpy as np
ai_math = AI()
y_true = [1, 0, 1]
y_pred = [0.9, 0.1, 0.8]
print(ai_math.mean_squared_error(y_true, y_pred))

Measures prediction accuracy.

🧠Step 2: Sigmoid Activation

Apply neural network activation.

x_values = np.array([-2, 0, 2])
print(ai_math.sigmoid(x_values))

Squashes numbers for AI decisions.

🗂️Step 3: Cluster Data

Group data with K-Means.

X_clusters = np.array([[1, 2], [1, 3], [5, 8]])
centroids, labels = ai_math.k_means(X_clusters, k=2)
print("Centroids:", centroids)

Groups points into clusters.

8. BasicMath: Everyday Calculations

Perform essential operations with BasicMath.

➕Step 1: Basic Arithmetic

Addition, subtraction, multiplication, division.

from clawpy import BasicMath
math_obj = BasicMath()
print(math_obj.add(5, 3))       # 8
print(math_obj.subtract(10, 4)) # 6
print(math_obj.multiply(6, 7))  # 42
print(math_obj.divide(10, 2))   # 5.0

Covers everyday arithmetic.

🧠Step 2: Power and Roots

Calculate powers and roots.

print(math_obj.power(2, 3))     # 8
print(math_obj.root(27, 3))     # 3.0

For exponentiation and roots.

🌀Step 3: Modulo and Factorial

Find remainders and factorials.

print(math_obj.modulo(10, 3))   # 1
print(math_obj.factorial(5))    # 120

For remainders and permutations.

9. Help: Common Issues & Solutions

Troubleshooting tips to keep your ClawPy experience smooth.

❓Issue 1: ModuleNotFoundError

Python can’t find ClawPy modules.

Solution: Add ClawPy to Python path.

import sys
sys.path.insert(0, "./")

Ensure ClawPy is in your directory.

🛠️Issue 2: Missing Dependencies

Errors from missing NumPy or SymPy.

Solution: Install dependencies.

pip install numpy sympy

Run in your terminal.

🖼️Issue 3: Tesseract Animation

Tesseract window won’t close.

Solution: Close via window or task manager.

Graphical outputs require manual closing.

📉Issue 4: Incorrect Outputs

Strange results from wrong units/formats.

Solution: Check input requirements.

See code comments for units.
